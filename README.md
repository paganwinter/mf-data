# Mutual Fund Data

### Funds List

- Open ended Growth Funds - https://paganwinter.github.io/mf-data/data/funds-filtered.json

### Fund NAVs

- https://paganwinter.github.io/mf-data/data/navs/{AMFI-code}.json
